# Wynnrot

Wynncraft QoL mods? Never heard of them, it's time for some stupid changes

<img src="https://raw.githubusercontent.com/ShadowCat117/ShadowCat117.github.io/refs/heads/main/Resources/Wynnrot/Queen67.gif" alt="A 67 Fruma Queen">

## Current Features
- 67 Fruma Queen
  - Replaces the Queen portrait with the 67 variant
- Eternal Hunger Sui
  - Replaces all of Sui's portraits with the eating portraits
- Meow
  - Plays a random Minecraft cat sound when a chat message contains the word "meow"
- Dancing Queen
  - Adds the Fruma Queen dancing to your screen
- Le Big Fishe
  - Replaces the "The Piranha" title in Galleon's Graveyard with "Le Big Fishe" and the same for the Corrupted version
- Rainbow Text
  - Render the custom Wynncraft HUD or all text with the rainbow colour
- Horse Death
  - Plays the horse death sound when you die. Gets progressively louder everytime you die in the current session
